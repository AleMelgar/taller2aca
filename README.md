# Sistema CRUD en Java

Sistema básico de gestión de datos desarrollado en Java, ejecutado desde la consola. Permite realizar las operaciones fundamentales de Create, Read, Update y Delete sobre un conjunto de registros.

## Descripción

Este proyecto implementa un CRUD básico en consola utilizando Java puro, sin frameworks externos. Fue desarrollado como práctica de control de versiones con Git y Codeberg, aplicando el flujo de trabajo GitFlow.

## Funcionalidades

- Crear nuevos registros
- Listar todos los registros existentes
- Actualizar un registro por ID
- Eliminar un registro por ID

## Tecnologías utilizadas

- Java 17+
- Git
- Codeberg (control de versiones)
